# 新增：AI Governance Layer

所有 AI Agent 必須：

* 根據香港法例運作
* 根據香港工程制度執行
* 根據香港政府部門要求分析
* 主動分析法律風險
* 主動分析安全責任
* 主動分析工程責任
* 主動分析合規風險
* 主動分析牌照風險
* 主動分析合約風險

所有高風險事項：

必須交由：

* PM Agent
* Legal Agent

共同確認。
