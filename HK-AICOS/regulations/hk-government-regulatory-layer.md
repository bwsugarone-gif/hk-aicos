# 香港政府部門法規 Layer - 總覽
## Hong Kong Government Regulatory Layer Overview

---

## 系統用途

本文件為 HK-AICOS 系統的**香港政府部門法規總覽**，列出所有相關香港政府部門及其法規要求。

所有 Agent 在處理任務時，必須：
1. 識別涉及哪些政府部門
2. 讀取相關部門的 Regulation Layer 文件
3. 檢查合規要求
4. 標示風險等級
5. 提示需要專業人士確認的事項

## Agent 使用指引

### 1. 識別涉及部門

Agent 處理任務時，首先識別涉及哪些政府部門：

```
任務輸入
    ↓
關鍵字識別
    ↓
識別涉及部門
    ↓
讀取相關 Regulation Layer
```

### 2. 檢查合規要求

讀取相關部門的 Regulation Layer 文件，檢查：
- 法定要求
- 申請程序
- 牌照要求
- 檢驗要求
- 通報要求

### 3. 風險評估

評估合規風險：
- 是否符合法定要求
- 是否需要申請批准
- 是否需要專業人士確認
- 是否需要通報政府部門

### 4. 輸出提示

在輸出中明確提示：
- 涉及哪些政府部門
- 需要遵守哪些法規
- 需要哪些專業人士確認
- 需要哪些申請或批准

## 法定責任提醒

### 重要原則

> ⚠️ **所有涉及香港政府部門法定要求的事項，必須由香港合資格專業人士確認。**
>
> AI Agent 只能提供初步分析及提示，不可取代專業人士的法定責任。

### 法定簽署文件

以下文件必須由香港合資格專業人士簽署：

| 文件類型 | 簽署人 | 部門 |
|---------|-------|------|
| 建築圖則 | 認可人士（AP） | BD |
| 結構圖則 | 註冊結構工程師（RSE） | BD |
| 電力圖則 | 註冊電業工程人員（REW） | EMSD |
| 消防圖則 | 註冊消防工程師 | FSD |
| 水務圖則 | 持牌水喉匠 | WSD |
| 環境評估 | 環境顧問 | EPD |

## 政府部門申請流程

### 一般流程

```
1. 準備申請文件
    ↓
2. 專業人士審查及簽署
    ↓
3. 提交政府部門
    ↓
4. 政府部門審批
    ↓
5. 獲得批准
    ↓
6. 開始施工
    ↓
7. 政府部門檢查
    ↓
8. 完工驗收
```

### 常見申請

| 申請類型 | 部門 | 審批時間（參考） |
|---------|------|----------------|
| 建築圖則 | BD | 60-90 天 |
| 小型工程 | BD | 即時-14 天 |
| 掘路許可 | HyD | 30-60 天 |
| 食水接駁 | WSD | 30-45 天 |
| 排水接駁 | DSD | 30-45 天 |
| 環境許可 | EPD | 60-90 天 |
| 電力接駁 | EMSD | 30-60 天 |

## 政府部門檢查

### 常見檢查類型

| 檢查類型 | 部門 | 檢查內容 |
|---------|------|---------|
| 工地安全檢查 | Labour Dept | 安全措施、PPE、工人牌照 |
| 建築工程檢查 | BD | 施工符合圖則、結構安全 |
| 電力工程檢查 | EMSD | 電力安全、持牌電工 |
| 環境檢查 | EPD | 噪音、廢物、空氣質素 |
| 消防系統檢查 | FSD | 消防設備、逃生路線 |
| 食水系統檢查 | WSD | 水質、水管安裝 |

### 檢查不合格處理

```
政府部門檢查
    ↓
發現不合格
    ↓
發出改善通知
    ↓
限期改善
    ↓
複檢
    ↓
合格 → 繼續施工
不合格 → 停工令 / 檢控
```

## 違規後果

### 常見違規及後果

| 違規類型 | 部門 | 可能後果 |
|---------|------|---------|
| 無牌施工 | BD | 停工令、檢控、罰款 |
| 安全違規 | Labour Dept | 停工令、檢控、罰款、監禁 |
| 電力違規 | EMSD | 停工令、檢控、罰款 |
| 環境違規 | EPD | 檢控、罰款 |
| 無掘路許可 | HyD | 檢控、罰款、賠償 |

### 嚴重後果

- 停工令
- 刑事檢控
- 罰款（可達數十萬至數百萬）
- 監禁（嚴重違規）
- 吊銷牌照
- 公司聲譽受損
