# WHATSAPP INPUT TEMPLATE
## HK-AICOS WhatsApp 輸入格式模板

---

**版本：** v2.0.0
**日期：** 2026-05-13
**狀態：** Phase 2.0 文件骨架

---

## 用途

本文件定義地盤人員透過 WhatsApp 向辦公室發送訊息的標準格式。

統一格式有助於：
- AI 快速分類任務
- 確保資訊完整
- 減少來回確認
- 提高處理效率

---

## 基本輸入格式

所有 WhatsApp 訊息必須包括以下欄位：

```
🏗️ HK-AICOS 地盤報告
━━━━━━━━━━━━━━━━━━━━
📋 類型：[任務類型]
🔢 項目：[項目編號]
📅 日期：[YYYY-MM-DD]
⏰ 時間：[HH:MM]
📍 地點：[地盤位置/樓層/區域]
🚨 緊急：[緊急/一般/低優先]
━━━━━━━━━━━━━━━━━━━━
📝 描述：
[詳細描述]
━━━━━━━━━━━━━━━━━━━━
📎 附件：[有/無]
[如有，描述附件內容]
━━━━━━━━━━━━━━━━━━━━
👤 報告人：[姓名/職位]
```

---

## 任務類型分類

| 類型代碼 | 任務類型 | 觸發 Agent |
|---------|---------|-----------|
| `SAFETY` | 安全事故 / 安全問題 | Safety Agent + Legal Agent |
| `PROGRESS` | 工程進度 / 工序問題 | Engineering Agent |
| `MATERIAL` | 材料問題 / 供應商問題 | Material Agent |
| `VO` | VO 申請 / 工程變更 | QS Agent + Legal Agent |
| `WORKER` | 工人問題 / 出勤問題 | Foreman Agent |
| `SURVEY` | 測量問題 / 放線問題 | Surveying Agent |
| `DRAWING` | 圖則問題 / 設計問題 | Drafting Agent |
| `LEGAL` | 法律問題 / 合約問題 | Legal Agent + PM Agent |
| `FINANCE` | 財務問題 / 薪金問題 | Accounting Agent |
| `DAILY` | 地盤日報 | Foreman Agent + Engineering Agent |
| `URGENT` | 緊急事項 | PM Agent（立即處理） |
| `OTHER` | 其他 | PM Agent（分類後處理） |

```

━━━━━━━━━━━━━━━━━━━━

- 受傷人員：[姓名/職位/人數]
- 受傷程度：[輕傷/中傷/重傷/死亡]
- 已送院：[是/否]
```

## 圖片 / 附件處理

### 圖片輸入

當附上圖片時，必須同時提供文字描述：

```
📸 圖片描述：
- 圖片內容：[描述圖片顯示的內容]
- 問題位置：[指出圖片中的問題位置]
- 需要分析：[希望 AI 分析什麼]
```

### PDF 文件輸入

當附上 PDF 時，必須提供摘要：

```
📄 文件摘要：
- 文件類型：[合約/報告/圖則/其他]
- 文件名稱：[文件名稱]
- 關鍵內容：[摘要關鍵內容]
- 需要分析：[希望 AI 分析什麼]
```

## 輸入質量要求

### 必須提供

- ✅ 任務類型
- ✅ 項目編號
- ✅ 日期時間
- ✅ 詳細描述
- ✅ 緊急程度
- ✅ 報告人姓名

### 盡量提供

- 📌 具體地點
- 📌 相關人員
- 📌 已採取措施
- 📌 附件描述

### 不可省略

- ❌ 不可只發圖片而無文字描述
- ❌ 不可只說「有問題」而無詳細描述
- ❌ 不可省略項目編號
- ❌ 不可省略緊急程度
