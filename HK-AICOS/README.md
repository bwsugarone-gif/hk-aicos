# HK-AICOS
## Hong Kong AI Construction Operating System
## 香港 AI 建築工程作業系統

---

## 系統簡介

HK-AICOS 係一套企業級香港 AI 建築工程作業系統，專為香港建築工程行業設計。

系統整合：
- Multi-Agent Construction AI System
- 香港工程法規 Layer
- 部門 Agent Prompt
- Skill Library
- SOP Layer
- RAG Knowledge Layer
- ERP / Database Layer
- Memory Layer
- Workflow Layer
- Dashboard Layer
- Contract / Legal Layer

## 核心原則

1. **香港法規優先** - 所有 Agent 必須根據香港法例運作
2. **安全第一** - 不可忽略任何安全責任
3. **專業確認** - 高風險事項必須由合資格專業人士確認
4. **PM + Legal 雙重確認** - 所有高風險決策必須由 PM Agent 及 Legal Agent 共同確認
5. **透明記錄** - 所有決策及風險必須記錄
