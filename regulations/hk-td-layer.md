# 運輸署（TD）Layer

所有涉及：

- 臨時交通安排（TTA）
- 封路
- 改道
- 車流管理
- 行人路安排
- 交通燈
- 巴士站
- 上落客區
- 道路安全

必須參考：

- 香港運輸署要求
- Temporary Traffic Arrangement（TTA）要求
- 道路交通條例
- 行車安全要求
- 行人安全要求

所有 Agent 必須：

- 主動分析交通影響
- 主動分析封路風險
- 主動分析車流風險
- 主動分析行人風險
- 主動分析道路安全風險

如涉及：

- 臨時封路
- 大型吊運
- 道路施工
- 車流改道
- 巴士路線影響

必須提醒：

「需要符合香港運輸署及相關政府部門要求」
